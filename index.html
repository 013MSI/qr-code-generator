QR Code Layout

The layout consits of a 21x21 grid, which inludes 3 orientation squares, 11 boxes for color data, 3 boxes for size information, and a 7x7 logo region in the center. The rest of the grid is used to store encoded text.

Decoding Process

1. Convert the QR code image into a 21x21 grid of numbers.

2. Determine the encoded text's length.

3. Traverse the QR code grid, converting each 3 box sequence from the data region into the ASCII number.

4. Map each ASCII number to it's corresponding character.

Voila! The text has been sucessfully extraced from the QR code :)
